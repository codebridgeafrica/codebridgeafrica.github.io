The Courses

5 Professional Tracks

Each track is a standalone 6-month programme. All share the same 3-phase structure. You can take multiple tracks sequentially to build a full professional skill set.

1

Full-Stack Web Development

Starting from GHS 799/month

Build complete web applications from front-end to back-end. Covers everything needed to work as a professional web developer.

Phase 1 · Wks 1–8
Front-End
HTML, CSS, JavaScript ES6+, responsive design, Git/GitHub
Phase 2 · Wks 9–16
Back-End
Node.js, Express, REST APIs, MySQL/PostgreSQL, authentication
Phase 3 · Wks 17–24
Capstone
Build & deploy a full-stack app — React front-end + Node back-end
2

Data Science & Analytics

Starting from GHS 799/month

Turn raw data into actionable insights. Master Python, SQL, and visualisation tools used by analysts across industries.

Phase 1 · Wks 1–8
Python for Data
Python, Pandas, NumPy, data cleaning, exploratory analysis
Phase 2 · Wks 9–16
SQL & Visualisation
SQL queries, joins, aggregations, Power BI / Matplotlib dashboards
Phase 3 · Wks 17–24
Analytics Project
Analyse a real business dataset — present dashboard to stakeholders
3

Cloud Computing (AWS / GCP)

Starting from GHS 899/month

Deploy, manage and scale applications in the cloud. Prepares you for AWS Cloud Practitioner or GCP Associate certification.

Phase 1 · Wks 1–8
Cloud Foundations
Cloud concepts, AWS/GCP services, IAM, EC2, S3, networking basics
Phase 2 · Wks 9–16
Deployment & DevOps
Docker, CI/CD pipelines, serverless functions, monitoring & alerts
Phase 3 · Wks 17–24
Cert Prep & Project
Deploy a real app to the cloud + exam prep for AWS/GCP cert
4

Cybersecurity Fundamentals

Starting from GHS 899/month

Protect systems, networks, and data. Covers essential security skills for IT professionals and prepares for CompTIA Security+ certification.

Phase 1 · Wks 1–8
Security Foundations
Threats & attacks, network security, encryption basics, Linux essentials
Phase 2 · Wks 9–16
Ethical Hacking
Penetration testing basics, vulnerability scanning, OWASP Top 10
Phase 3 · Wks 17–24
Security Audit Project
Conduct a mock security audit + CompTIA Security+ exam prep
5

Python for Professionals

Starting from GHS 699/month

For non-developers who need coding skills for their job — finance, operations, HR, marketing. Automate repetitive tasks and work smarter with data.

Phase 1 · Wks 1–8
Python Basics
Variables, loops, functions, file handling, working with spreadsheets
Phase 2 · Wks 9–16
Automation & Data
Excel/CSV automation, Pandas, sending emails, API integrations
Phase 3 · Wks 17–24
Workplace Project
Automate a real task from your own job — measurable time saved
