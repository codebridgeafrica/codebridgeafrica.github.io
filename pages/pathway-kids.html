The Journey

4 Learning Stages

Children progress through four stages — each a 6-month course made up of 3 phases of 8 weeks. Every stage builds on the last. No rush, no pressure.

1

Digital Explorers Ages 6–7

Computers, internet safety, and block-based coding with Scratch. Students build their first interactive story.

Phase 1 · Wks 1–8
Computers & the Internet — how devices work, staying safe online
Phase 2 · Wks 9–16
Scratch Basics — sprites, backdrops, events and sequences
Phase 3 · Wks 17–24
My First Story — build a full interactive animated story in Scratch
2

Code Builders Ages 8–9

Deeper Scratch projects, introduction to loops, conditionals, and basic game development.

Phase 1 · Wks 1–8
Loops & Conditions — repeat, if/else, variables in Scratch
Phase 2 · Wks 9–16
Game Logic — scores, timers, collision detection
Phase 3 · Wks 17–24
My First Game — design, build and present a working Scratch game
3

Python Juniors Ages 10–11

First steps in real Python programming — variables, functions, and simple programmes.

Phase 1 · Wks 1–8
Python Basics — print, variables, input, data types
Phase 2 · Wks 9–16
Control & Functions — loops, conditionals, defining functions
Phase 3 · Wks 17–24
My Python App — build a text-based quiz or calculator
4

Project Launch Age 12

Capstone stage — students design and build their own app or game, then present it at our showcase day.

Phase 1 · Wks 1–8
Design Thinking — ideation, wireframing, user stories
Phase 2 · Wks 9–16
Build Sprint — develop the project with instructor support
Phase 3 · Wks 17–24
Showcase Day — present to parents, peers and a panel

Skills & Outcomes

Computational Thinking

Breaking problems into steps, spotting patterns, and designing solutions — skills that apply to maths, science, and everyday life.

Digital Citizenship

Online safety, privacy awareness, responsible use of technology, and understanding the digital world they live in.

Creative Confidence

Building things from scratch instils confidence. Children learn that technology is something they can shape — not just consume.
