Course Stages

4 Progressive Courses

Each course is a 6-month journey split into 3 structured phases of 8 weeks. Teens can start at any stage based on prior experience.

1

Tech Foundations

GHS 399/month

The entry point. No experience needed. Teens learn how computers work, build their first websites, and write their first Python scripts.

Phase 1 · Wks 1–8
Digital Foundations
How computers work, internet basics, file management, intro to HTML
Phase 2 · Wks 9–16
Web & Python Intro
HTML/CSS web pages, Python basics — variables, loops, functions
Phase 3 · Wks 17–24
Mini Project
Build a personal website + a Python quiz app — present to the class
2

Web Development

GHS 549/month

Build real websites and web apps. Teens learn JavaScript, responsive design, and how to deploy a live site on the internet.

Phase 1 · Wks 1–8
CSS & Layout
CSS Grid, Flexbox, responsive design, media queries, UI principles
Phase 2 · Wks 9–16
JavaScript
DOM manipulation, events, fetch API, working with JSON data
Phase 3 · Wks 17–24
Live Project
Build & deploy a multi-page responsive website — goes on GitHub portfolio
3

Python & Data

GHS 549/month

Intermediate Python — algorithms, file handling, working with data, and intro to data visualisation.

Phase 1 · Wks 1–8
Intermediate Python
OOP basics, file I/O, error handling, modules & libraries
Phase 2 · Wks 9–16
Data Basics
CSV/spreadsheet data, Pandas intro, Matplotlib charts
Phase 3 · Wks 17–24
Data Project
Analyse a real Ghanaian dataset — produce charts and present findings
4

Capstone & Portfolio

GHS 549/month

The graduation stage. Teens choose a real-world problem, apply everything they've learned, and build a full solution presented to an audience.

Phase 1 · Wks 1–8
Research & Planning
Problem definition, research, tech stack selection, project plan
Phase 2 · Wks 9–16
Build Sprint
Full project development with weekly mentorship check-ins
Phase 3 · Wks 17–24
Demo Day
Present live project to parents, peers, and industry guests
